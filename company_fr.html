À propos du site

Conditions d'utilisation

Ce site web a été créé à partir d'un modèle.

Les liens, etc. affichés sur ce site web sont des effets personnels du créateur, mais le modèle est distribué.
Veuillez ne pas analyser les données du site web et créer la même chose.

Mise en page pour chaque appareil

Il est recommandé de l'utiliser sur une tablette ou un smartphone, mais vous pouvez également effectuer une vérification temporaire avec le dernier navigateur (IE à partir d'IE10). La largeur du navigateur se rétrécit et la mise en page est adaptée à la taille de chaque terminal.

Environnement pour les petits appareils (※largeur inférieure à 900px)

Le menu principal sera pliable (avec une icône à 3 barres).

Précautions sur ce site web

Attention : Veuillez ne pas utiliser abusivement les œuvres sur le site web (y compris les modèles et les sites créés par le créateur).

En ce qui concerne le modèle, comme indiqué ci-dessus, veuillez ne pas copier et coller l'URL du site web, etc. sur le site web et le créer vous-même. Si vous le voyez, veuillez le signaler.

À propos des erreurs sur le site web

Nous nous efforçons de corriger les erreurs lors de la création, mais si vous en trouvez, veuillez contacter le compte DISCORD du créateur.
Veuillez également nous contacter si vous avez des demandes.

Si un message d'avertissement s'affiche lors de la vérification dans l'aperçu (pour certains navigateurs)

Il s'agit principalement d'un avertissement qui apparaît avec le fichier javascript (fichier js), mais il n'apparaît pas sur le WEB. De plus, pendant que cet avertissement s'affiche, vous ne pouvez pas voir l'effet, alors veuillez cliquer dans le message d'avertissement pour l'annuler. Cela vous permettra de voir l'effet correctement.
